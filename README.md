# AI Tools Usage Analysis Among Indian College Students

## Overview

This project presents an Exploratory Data Analysis (EDA) of AI tool usage among Indian college students. The aim is to understand how students use AI tools in their learning activities, identify popular tools, and explore usage patterns, awareness levels, and their impact on academics.

## Objectives

* Understand AI adoption among college students.
* Identify the most commonly used AI tools.
* Analyze student usage patterns and preferences.
* Explore the relationship between AI usage and academics.
* Study awareness and trust levels related to AI tools.

## Dataset

The dataset contains responses from **496 Indian college students** and includes information such as:

* Academic stream
* Year of study
* Preferred AI tool
* Daily AI usage
* Awareness level
* Trust in AI tools
* Academic impact and learning support

## Tools and Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Analysis Performed

The project includes:

* Data cleaning and preprocessing
* Missing value analysis
* Distribution analysis
* AI tool popularity analysis
* Usage pattern analysis
* Correlation analysis
* Data visualization and insight generation

## Key Insights

* AI tools are widely used among college students.
* ChatGPT was one of the most frequently used AI tools in the dataset.
* Students use AI tools mainly for learning, assignments, and academic support.
* Usage patterns vary across different academic backgrounds.
* Awareness and trust levels influence AI adoption and usage.

## Conclusion

The analysis shows that AI tools have become an important part of the learning experience for many students. By exploring usage patterns and preferences, this project provides insights into how students interact with AI technologies and highlights the growing role of AI in education.
